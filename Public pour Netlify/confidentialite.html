2. Données collectées

Dans le cadre de l'utilisation de SOS Storytelling, nous collectons les données suivantes :

2.1 Données d'identification

  • Adresse email
  • Nom et prénom (optionnel)
  • Nom de l'organisation

2.2 Données d'utilisation

  • Contenus générés (posts, newsletters, emails)
  • Profils de voix et styles d'écriture
  • Frameworks et prompts personnalisés
  • Progression et statistiques d'utilisation
  • Messages échangés via le chat coach

2.3 Données techniques

  • Adresse IP
  • Type de navigateur
  • Logs de connexion

4. Garanties importantes

Vos contenus vous appartiennent.

  • Vos contenus et profils de voix ne sont jamais utilisés pour entraîner des modèles d'IA externes
  • Vos frameworks et prompts restent confidentiels
  • Nous ne vendons jamais vos données à des tiers

5. Destinataires des données

Vos données peuvent être partagées avec :

  • Supabase (hébergement base de données) - UE
  • Netlify (hébergement web) - USA (clauses contractuelles types)
  • Anthropic (API Claude) - vos propres clés API
  • Perplexity (API recherche) - vos propres clés API
  • Lemon Squeezy (paiement) - pour les transactions uniquement

7. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer à certains traitements
  • Droit à la limitation : restreindre le traitement

Pour exercer ces droits : contact@myinnerquest.fr

Délai de réponse : 1 mois maximum.

8. Cookies

SOS Storytelling utilise uniquement des cookies techniques essentiels :

  • Cookies d'authentification (session utilisateur)
  • Cookies de préférences (thème, langue)

Nous n'utilisons aucun cookie publicitaire ou de tracking tiers.

9. Sécurité

Nous mettons en œuvre les mesures suivantes pour protéger vos données :

  • Chiffrement des données en transit (HTTPS/TLS)
  • Authentification sécurisée via Supabase Auth
  • Accès aux données limité au personnel autorisé
  • Hébergement sur des serveurs sécurisés (Supabase, Netlify)

10. Transferts internationaux

Certaines données peuvent être transférées vers les États-Unis (Netlify, Anthropic, Perplexity). Ces transferts sont encadrés par :

  • Les clauses contractuelles types de la Commission européenne
  • Le Data Privacy Framework pour les entreprises certifiées
