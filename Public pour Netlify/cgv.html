Article 1 - Objet

Les présentes Conditions Générales de Vente (CGV) régissent les relations contractuelles entre :

Toute souscription implique l'acceptation sans réserve des présentes CGV.

Article 2 - Description du service

SOS Storytelling est une application web de création de contenu assistée par intelligence artificielle, comprenant :

  • Génération de contenus pour réseaux sociaux (posts, carrousels, scripts vidéo, etc.)
  • 20+ structures narratives (AIDA, PAS, Storytelling, etc.)
  • Fonctionnalité "Ma Voix" d'analyse et reproduction du style d'écriture
  • Planning IA pour la planification éditoriale
  • Exports vers outils tiers (Metricool, Swello, Buffer, CSV)
  • Mode multi-clients pour les agences (selon le plan souscrit)

Un frais de setup initial de 1 500 € HT est facturé une seule fois lors de la mise en place. Il comprend :

  • Configuration white label (couleurs, logo, nom de l'app)

  • Accompagnement à la configuration des clés API

3.3 Coûts API externes

Le Client utilise ses propres clés API pour les services d'intelligence artificielle (Anthropic Claude, Perplexity). Ces coûts sont payés directement aux fournisseurs, estimés entre 100 et 300 € par mois selon l'usage.

3.4 Démonstration

Une démonstration personnalisée de 30 minutes est proposée avant toute souscription. Cette démo permet de voir l'outil avec vos couleurs et votre cas d'usage, sans engagement.

Article 4 - Souscription et paiement

4.1 Processus de souscription

La souscription s'effectue en ligne via notre prestataire de paiement Lemon Squeezy. Le Client doit :

  1. Choisir son plan
  2. Créer un compte avec une adresse email valide
  3. Renseigner ses informations de paiement
  4. Valider sa commande

Le contrat est formé dès la confirmation du paiement.

4.2 Moyens de paiement

Les paiements sont traités par Lemon Squeezy. Les moyens de paiement acceptés incluent :

  • Carte bancaire (Visa, Mastercard, American Express)
  • PayPal

4.3 Facturation

L'abonnement est facturé mensuellement, à date anniversaire de la souscription. Une facture est envoyée automatiquement par email après chaque paiement.
